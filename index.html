Ablation Studies

Latent-Space Formulation: WanVAE vs. FluxVAE  Pour Water — OoD

WanVAE applies 4× temporal compression coupling consecutive frames in latent space. FluxVAE encodes each frame independently (1× temporal compression).

WanVAE — ours
OoD PSNR 24.57 dB  ·  SSIM 0.875
FluxVAE — frame-independent
OoD PSNR 24.09 dB  ·  SSIM 0.875

Action Dimensionality: 3-DoF vs. 8-DoF  Cloth Move — OoD

3-DoF: shared Δxyz displacement for both arms (benchmark default). 8-DoF: full per-arm pose + grasp signal. Richer conditioning dramatically improves OoD generalization.

3-DoF — benchmark
OoD PSNR 16.23 dB
8-DoF — full per-arm
OoD PSNR 22.82 dB  (+6.59 dB)

Action Conditioning: AdaLN vs. Cross-Attention  Robot Arm — OoD

AdaLN injects action as a global scale-shift per block. Cross-attention injects per-timestep action tokens. Gains are larger for high action-dim tasks.

AdaLN — default
OoD PSNR 21.83 dB
Cross-Attention
OoD PSNR 23.38 dB  (+1.55 dB)
