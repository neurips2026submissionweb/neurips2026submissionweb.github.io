ACWM-Phys: Investigating Generalized Physical Interaction in Action-Conditioned Video World Models

Anonymous Authors — NeurIPS 2026 Submission
This page contains video results that supplement the PDF submission.
We introduce ACWM-Phys, a benchmark for evaluating action-conditioned video world models across four physically diverse interaction regimes (rigid-body, deformable, particle, and kinematics), and ACWM-DiT, a latent diffusion transformer baseline trained with flow matching. Each video below shows ground truth (left) alongside the model prediction (right).

Ablation Studies

Latent-Space Formulation: WanVAE vs. FluxVAE  Pour Water — OoD

WanVAE applies 4× temporal compression, coupling consecutive frames in the latent space. FluxVAE encodes each frame independently. Both evaluated on out-of-distribution water quantities.

WanVAE (ours)
PSNR 24.57 dB  ·  SSIM 0.875
FluxVAE (frame-independent)
PSNR 24.09 dB  ·  SSIM 0.875

Action Dimensionality: 3-DoF vs. 8-DoF  Cloth Move — OoD

3-DoF uses a shared Δxyz displacement for both arms. 8-DoF provides full per-arm pose + grasp signals. Evaluated on out-of-distribution cloth sizes.

3-DoF actions (benchmark)
OoD PSNR 16.23 dB
8-DoF actions (full per-arm)
OoD PSNR 22.82 dB  (+6.59 dB)

Action Conditioning: AdaLN vs. Cross-Attention  Robot Arm — OoD

AdaLN injects action as a global scale-shift at each block. Cross-attention injects per-timestep action tokens via dedicated attention layers. Evaluated on out-of-distribution workspace regions.

AdaLN (default)
OoD PSNR 21.83 dB
Cross-Attention
OoD PSNR 23.38 dB  (+1.55 dB)
